# TaskProfileAPI

A simple Laravel API for managing Users, Profiles, and Tasks with Sanctum authentication.

## Features

- User Registration
- User Login with Token Authentication (Laravel Sanctum)
- User Logout
- CRUD operations for Tasks (per user)
- CRUD operations for Profile (per user)
- API protection using middleware and ownership checks

## Models

- User
- Profile
- Task

## Requirements

- PHP >= 8.1
- Laravel >= 10
- MySQL or any supported database
- Composer
